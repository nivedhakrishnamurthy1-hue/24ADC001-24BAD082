# BI Lab - Experiment 1: Explore BI Tool Interface

## Objective
Explore Tableau and Power BI interfaces using a GOOG stock dataset,
build basic visualizations, and combine them into an interactive dashboard.

## Dataset
GOOG historical stock data (2016-2021), sourced from Kaggle, with a
Predicted Price column added using a 3-day Simple Moving Average.

## Contents
- GOOG_full_with_predicted.csv - dataset used
- GOOG_Stock_Dashboard.twbx - Tableau workbook
- GOOG_Stock_Dashboard.pbix - Power BI file
- Screenshots of both finished dashboards

## Charts Built
- Line chart: Close Price trend over time
- Scatter plot: Trading Volume vs Predicted Close
- Highlight table / Table: Predicted Price by year
- KPI: Average Prediction Error
